Xandev's Awesome Graphical User Interface
====

XAGUI is an Open Source, multi-platform* GUI library written in C++.

Available Controls:
- Label
- Button
- Window
- Slider
- Text Box
- Password Text Box
- Multiline Text Box
- Check Box
- Radio Button
- Picture Box
- Proggress Bar
- Analog Stick

Features:
- Open Source
- Multi-platform
- Wiki Documentation

*Multi-platform - when renderer support it. XAGUI originally was written based on SDL2.
