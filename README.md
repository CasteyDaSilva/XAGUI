Xandev's Awesome Graphical User Interface
====

XAGUI is an Open Source, multi-platform GUI library written in C++ and based on SDL2.

Available Controls:
- Label
- Button
- Window
- Slider
- Text Box
- Password Text Box
- Multiline Text Box
- Check Box
- Radio Button
- Picture Box
- Proggress Bar
- Analog Stick

Features:
- Open Source
- Multi-platform
- Wiki Documentation
